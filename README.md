# ☁️ Projeto AWS: Otimização de Custos - Abstergo Industries

Este repositório contém o relatório de implementação estratégica de serviços AWS para a **Abstergo Industries**, com foco em redução de custos operacionais e modernização de infraestrutura.

## 📌 Objetivo

O objetivo deste projeto foi selecionar e documentar três serviços da AWS que proporcionam economia imediata e maior eficiência para uma empresa do setor farmacêutico, utilizando conceitos de computação serverless e otimização de armazenamento.
